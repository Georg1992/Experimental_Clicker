BELARUS CHAMP TOOLS
=====================

Install
-------

Extract this ZIP, then:

1. Double-click Install.cmd
   (click Yes if Windows asks for permission)

2. Restart your PC if the installer asks you to
   (only needed the first time)

3. Double-click Belarus Champ Tools.exe in this folder
   Click Start, configure your tools, then open your game.
   Press End to toggle start/stop.


Uninstall
---------

1. Double-click Uninstall.cmd
   (click Yes if Windows asks for permission; no window should appear)

2. Restart your PC if asked

3. Delete this folder
