BELARUS CHAMP CLICKER
=====================

Установка
---------

Распакуйте ZIP, затем:

1. Дважды щёлкните Install.cmd
   (нажмите «Да», если Windows запросит разрешение)

2. Перезагрузите компьютер, если установщик попросит
   (нужно только при первой установке)

3. Дважды щёлкните Belarus Champ Clicker.exe в этой папке
   Нажмите Start, добавьте клавишу-триггер, затем запустите игру.
   Удерживайте триггер для кликов. End — пауза.


Удаление
--------

1. Дважды щёлкните Uninstall.cmd
   (нажмите «Да», если Windows запросит разрешение; окно не должно появляться)

2. Перезагрузите компьютер, если потребуется

3. Удалите эту папку
